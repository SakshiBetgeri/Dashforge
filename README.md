# 🔮 DashForge — AI Dashboard Builder

> Transform any CSV, Excel, PDF or image into beautiful AI-powered dashboards instantly.

---

## 🚀 How to Run

**1. Clone the repo**
```bash
git clone https://github.com/SakshiBetgeri/dashforge.git
cd dashforge
```

**2. Install dependencies**
```bash
npm install
```

**3. Start the app**
```bash
npm run dev
```

**4. Open browser**
```
http://localhost:3000
```

---

## ✨ Features

- **Multi-format Upload** — CSV, Excel, PDF and image files
- **Auto Chart Generation** — intelligent chart type selection based on data
- **KPI Cards** — key metrics calculated automatically from actual data
- **Gemini AI Insights** — anomaly detection, trend analysis and opportunity identification
- **AI Chat Assistant** — ask any question about your data in plain English
- **Smart Filters** — dynamic filters that update all charts in real time
- **Export Options** — download filtered CSV and copy AI report instantly
- **Domain Agnostic** — works with healthcare, sales, HR, education, finance and any domain
- **Bento Grid UI** — glassmorphism dark design system
- **Responsive** — works across all device sizes

---

## 🧠 AI Features

### Insight Cards
Three tier AI analysis powered by Gemini:
- 🔴 **Anomaly** — unusual patterns or urgent issues
- 🟡 **Trend** — directional patterns over time
- 🟢 **Opportunity** — highest value actionable findings

### AI Chat Assistant
Ask questions in plain English:
- "What is the average score?"
- "Which category performs best?"
- "Find anomalies in the data"
- "Give me a summary"

| Layer | Technology |

## 🚀 Netlify Deployment

1. Push code to GitHub
2. Connect repo to Netlify
3. Set build settings:
   - Build Command: `npm run build`
   - Publish Directory: `dist`
4. The included `netlify.toml` handles SPA routing automatically
